  1. Loops are commands that tell the computer to repeat a block of code a certain number of times. The exact number will depend on the command and how you write it.
  2. for loops
    The "template" of a for command is:
    for loop_variable in range(...): loop_body
    The loop variable is just like a regular variable: it stores values. This variable may (or may not) be used in the loop body. This body is a sequence of one or more commands.
    Think of range(...) as a list of values. On the first iteration, the loop variable stores the first value in this list. On the second iteration it stores the second values in the list. And so on and so forth. Everytime the code is repeated, the value stored in the loop variable changes.
    # range(n) is the list of integers from 0 to n-1 def forExample1(n): for i in range(n): print(i, end=" ") forExample1(8)
    # range(m,n) is the list of integers from m to n-1 def forExample2(m, n): for i in range(1,n): print(i, end=" ") forExample2(2, 8)
    # range(m,n,s) is the list of every s integer from m to n-1 def forExample3(m, n, s): for i in range(m, n, s): print(i, end=" ") forExample3(1, 51, 10)

    Accumulating results in a variable
    def factorial(n): x = 1 for i in range(1, n+1): # Take the value in x, multiply by the value in i and store in x x = x * i return x print( factorial(5) )

    IMPORTANT: do not change the value of the loop variable inside the loop!
  3. while loops
    The "template" of a while command is:
    while condition: loop_body
    This is read as: while the condition is true, repeat the loop body. ATTENTION: if the condition is always true, your code will run forever (it loops infinitely). So you better check that the condition will become false eventually!
    def whileExample(n): while n > 0: n = n - 1 print(n, end=" ") return whileExample(17)

  4. Controlling loops
    A loop can be artificially stopped by using the command break.
    def sumWithLimit(n, limit): sum = 0 for i in range(n): sum = sum + i if sum > limit: sum = limit break return sum print( sumWithLimit(100, 2897) ) print( sumWithLimit(100, 10000) )

    One loop iteration can be skipped using the command continue.
    def sumIfMultiple(m, n, d): sum = 0 for i in range(m, n): if (i % d != 0): continue sum = sum + i return sum print( sumIfMultiple(4, 20, 5) ) # 5 + 10 + 15 = 30 print( sumIfMultiple(7, 17, 4) ) # 8 + 12 + 16 = 36
