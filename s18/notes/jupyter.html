--- title: Intro to Jupyter Notebook layout: notes --- Jupyter Notebook is an app that runs on the browser for editing and running notebook documents. According to Jupyter's website:
Notebook documents (or "notebooks", all lower case) are documents produced by the Jupyter Notebook App, which contain both computer code (e.g. python) and rich text elements (paragraph, equations, figures, links, etc.). Notebook documents are both human-readable documents containing the analysis description and the results (figures, tables, etc.) as well as executable documents which can be run to perform data analysis,
Even though it runs on the browser, it is not on the internet! You will need to install it on your computer. Depending on how you installed python, you may already have it. To check that, see if you can find Jupyter on your applications, or open the terminal (or cmd for Windows) and type jupyter notebook. If that does not work and you need to install it, go here. Some useful things to know:
  • Each box after a In []: tag is called a cell.
  • Cells can contain python code or explanatory text.
  • The explanatory text can be typed using markdown text. That's the same syntax used on wiki pages, for example. This is a nice cheatsheet with markdown commands to make your text look nice.
  • There are two main keyboard shortcuts to run the code or display the text in a cell: ctrl + Enter and shift + enter. Using the shift one creates a new cell.
  • Jupyter has two modes: a command mode (when cells cannot be edited and trying to type something down does weird things), and an edit mode (when cells can be edited). In command mode the left margin of the selected cell is blue, in edit mode it is green. You can switch modes by clicking in or out of a cell, or by typing esc and enter.
  • Keyboard shortcuts are our friends. Once we get used to them, they save us a lot of time! To see what shortcuts are available in Jupyter, go to Help > Keyboard Shortcuts.
  • To reset a notebook (delete variables, close files, etc.) and start it from scratch, go to Kernel > Restart.
  • Googling things like "What is the markdown syntax for ...?" and "How do I ... in Jupyter?" is ok.
Why notebooks?
Jupyter notebook is a nice environment for data analysis because we can, in one place, play around with the code, see graphs and document our analysis. Moreover, we can share our findings easily in a nicely formated way without having to worry about importing/copying things from one software to the other. People in the scientific community use it all the time, and so should you since you are on your path to becoming a scientist :)
