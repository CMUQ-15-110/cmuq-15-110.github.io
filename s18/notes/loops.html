Loops are commands that tell the computer to repeat a block of code a certain number of times. The exact number will depend on the command and how you write it.

  1. for loops

    The "template" of a for command is:
    # for loop_variable in range(...): # loop_body

    The loop variable is just like a regular variable: it stores values. This variable may (or may not) be used in the loop body. This body is a sequence of one or more commands.

    Think of range(...) as a list of values. On the first iteration, the loop variable stores the first value in this list. On the second iteration it stores the second values in the list. And so on and so forth. Everytime the code is repeated, the value stored in the loop variable changes.

    # range(n) is the list of integers from 0 to n-1 def forExample1(n): for i in range(n): print(i, end=" ") forExample1(8)

    # range(m,n) is the list of integers from m to n-1 def forExample2(m, n): for i in range(1,n): print(i, end=" ") forExample2(2, 8)

    # range(m,n,s) is the list of every s integer from m to n-1 def forExample3(m, n, s): for i in range(m, n, s): print(i, end=" ") forExample3(1, 51, 10)

    2. IMPORTANT: never change the value of the loop variable inside the loop!

  2. Wow

    Given a number n write a program that prints 'Wo...ow' with n o's.
    def wow(n): print("W", end="") for i in range(n): print("o", end="") print("w") return wow(3) wow(6)


  3. Accumulating results in a variable

    Compute the factorial.
    def factorial(n): x = 1 for i in range(1, n+1): # Take the value in x, multiply by the value in i and store in x x = x * i return x print( factorial(5) )


    Sum the numbers from m to n.
    def sumMtoN(m, n): sum = 0 for i in range(m, n+1): sum += i # Same as sum = sum + i return sum print( sumMtoN(1, 3) ) print( sumMtoN(10, 15) )


  4. Nested loops

    Print an n x m board with a * at each position.
    def board(n, m): for i in range(n): for j in range(m): print("* ", end="") print() return board(3, 3) print() # Blank line in the middle board(4, 6) print() # Blank line in the middle board(6, 4)

    Given n print a n x n board where the main diagonal is composed of O characters.
    def boardDiagonal(n): for i in range(n): for j in range(n): if i == j: print("O", end=" ") else: print("*", end=" ") print() return boardDiagonal(6) print() # Blank line in the middle boardDiagonal(10)

    Given n print a n x n board where each position (i,j) where i is a multiple of j or vice-versa is marked with an O.
    def boardMultiple(n): for i in range(1,n+1): for j in range(1,n+1): if i % j == 0 or j % i == 0: print("O", end=" ") else: print("*", end=" ") print() return boardMultiple(6) print() # Blank line in the middle boardMultiple(10) print() # Blank line in the middle boardMultiple(40)

  5. while loops

    The "template" of a while command is:
    # while condition: # loop_body

    This is read as: while the condition is true, repeat the loop body.
    ATTENTION: if the condition is always true, your code will run forever (it loops infinitely). So you better check that the condition will become false eventually!

    def whileExample(n): while n > 0: n = n - 1 print(n, end=" ") return whileExample(17)


  6. Controlling loops

    A loop can be artificially stopped by using the command break.
    def sumWithLimit(n, limit): sum = 0 for i in range(n): sum = sum + i if sum > limit: sum = limit break return sum print( sumWithLimit(100, 2897) ) print( sumWithLimit(100, 10000) )

    One loop iteration can be skipped using the command continue.
    def sumIfMultiple(m, n, d): sum = 0 for i in range(m, n): if (i % d != 0): continue sum = sum + i return sum print( sumIfMultiple(4, 20, 5) ) # 5 + 10 + 15 = 30 print( sumIfMultiple(7, 17, 4) ) # 8 + 12 + 16 = 36

    break and continue work in while loops as well.

  7. Prime numbers

    A number is called prime if it is only divisible by 1 and itself. Using loops, we can write a code that checks if a number is prime or not.
    def isPrime(n): if n == 1: return False d = 2 while d < n: if n % d == 0: return False d = d + 1 return True print( isPrime(1) ) print( isPrime(2) ) print( isPrime(3) ) print( isPrime(4) ) print( isPrime(5) ) print( isPrime(16) ) print( isPrime(17) )


  8. Epidemic

    Suppose there was a breakout of a very contagious epidemic disease. Scientists suspect that every sick person contaminates, on average, 3 other people on the first day. Write a program that computes the number of contaminated people on the nth day.
    def countContaminated(n): total_contaminated = 1 # one sick person on the first day total_contagious = 1 # this first person is contagious for i in range(1,n+1): # On the next day, each person contaminates 3 others, who become the new contagious people total_contagious = total_contagious * 3 # These new contagious people are count towards the number of total contaminated total_contaminated = total_contaminated + total_contagious return total_contaminated print( countContaminated(2) ) print( countContaminated(7) ) print( countContaminated(30) ) print( countContaminated(365) )

    If we want to know how many days it will take until everyone in the world is contaminated, we don't need to keep testing with the program above. We can write another program that will do that for us!
    def allContaminated(): population = 7600000000 total_contaminated = 1 total_contagious = 1 days = 0 while (total_contaminated < population): total_contagious = total_contagious * 3 total_contaminated = total_contaminated + total_contagious days += 1 return days print(allContaminated())
